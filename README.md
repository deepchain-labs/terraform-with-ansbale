# terraform-with-ansbale
## Prerequisite installation
* Aws Cli (https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html)
* Ansible 
* terraform
## Create aws secret key
Go to > `` Create Credential `` and create access key
